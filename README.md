# Five SDK Tutorial
> Javascript esModule with react hoc

## 系统要求

- node >= 12.x
- 一个现代浏览器:

| Safari | Safari on iOS | Chrome  | Chrome for Android | Edge | Firefox |
|:------:|:-------------:|--------:|:------------------:|:----:|:-------:|
| >= 9   | >= 9          | >= 49   | >= 93              | >= 13| >= 45   |

## 安装依赖

```shell
npm install
```

## 本地运行

```shell
npm run dev
```
